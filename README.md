# SVN DiffTo 

* right click on project in workspace and choose "SVN DiffTo"
* type svn tag (default ^/trunk)
* wait to generate

## Features

* shows svn diff between local project and any svn tag within same repository

## Requirements

* Ubuntu 20.04
