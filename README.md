# SVN DiffTo 

* ctrl+shift+p
* SVN DiffTo
* choose project
* type svn tag (default ^/trunk)
* wait to generate

## Features

* shows svn diff between local project and any svn tag within same repository

## Requirements

* Ubuntu 20.04
